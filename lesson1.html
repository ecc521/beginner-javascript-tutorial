Taking action based on user input

//Get a value from the user
value = prompt("Please type in a number")
if (value < 0) {
  alert("You picked a negative number!") //Alert the user they picked a negative number
}
else if (value < 10) {
  alert("You picked a small number") //Alert the user they picked a small number
}
else if (value < 100) {
  alert("You picked a medium number") //Alert the user they picked a medium number
}
else {
  alert("You picked a large number") //Alert the user they picked a large number
}
document.write("You picked the number " + value)

Consider the following code:

This program asks the user for a number, and assigns the users input to a variable. It then proceeds to call the number either negative, small, medium, or large.

There isn't much special about this program - but there is something special about JavaScript - it can run in the browser.
